QubeFace JS

Create 3D cubes easily!

QubeFace is a free JavaScript library to create easily CSS 3D cubes for forms, images or other web content.
This library uses the MIT license and has no other dependencies.

QubeStar / QubeControl

Spin and swipe your cubes!

QubeStar is a free JavaScript library to optimally spin and swipe your 3D cube. The QubeStar library contains QubeControl and QubeItem. QubeControl and QubeItem's are for paging over items with spinning and swiping gesture.
This library uses the MIT license and just needs qubestar.js, but has no other dependencies.
